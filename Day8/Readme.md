Salesforce Summer Program – Day 8

1. What is LWC?

Lightning Web Components (LWC) is a modern UI framework used in Salesforce to build fast and reusable user interfaces.

LWC uses:

- HTML
- JavaScript
- CSS
- Metadata XML

It helps developers create responsive and component-based applications.

---

2. Why Salesforce Uses LWC

Salesforce uses LWC because it:

- Improves performance
- Supports reusable components
- Uses modern web standards
- Makes UI development faster
- Creates better user experience

LWC is more efficient and modern compared to older UI frameworks.

5. Frontend vs Backend Logic

Frontend/UI| Backend/Apex
Button click handling| Data validation
Form input display| Fee calculation
Notification display| Database operations
UI updates| Business logic
Navigation| Trigger execution

Frontend focuses on user interaction, while backend handles processing and data management.
