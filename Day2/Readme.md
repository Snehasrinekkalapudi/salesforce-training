# 📌 What is Salesforce Platform?

Salesforce Platform is a cloud-based platform used to build, customize, and manage business applications.  
It helps organizations automate business processes, manage customer data, and create secure applications without requiring heavy infrastructure setup.

---

# ✨ Features of Salesforce Platform

- Cloud-based environment
- Application development
- Data management
- Automation tools
- Security and scalability
- Low-code and no-code development

---

# 📖 Explain the Following Concepts

## 1️⃣ App

An App in Salesforce is a collection of tabs, objects, and features grouped together for a specific business purpose.

### ✅ Examples
- Sales App
- Service App
- Hospital Management App

Apps help users access related functionalities in one place.

---

## 2️⃣ Object

An Object in Salesforce is similar to a database table that stores information.

| Object | Purpose |
|---|---|
| Account | Stores company/customer information |
| Contact | Stores customer contact details |
| Lead | Stores potential customer information |

Objects contain records and fields.

---

## 3️⃣ Tab

A Tab is used to access objects, applications, dashboards, or other features in Salesforce.

Tabs make navigation easy for users.

### ✅ Examples
- Accounts Tab
- Contacts Tab
- Reports Tab

---

# ⚙️ Difference Between Configuration and Coding

| Configuration | Coding |
|---|---|
| Uses clicks and settings | Uses programming languages |
| No coding knowledge required | Coding knowledge required |
| Faster development | More customization possible |
| Uses tools like Flow and Process Builder | Uses Apex and Lightning Web Components |
| Easy to maintain | Requires developer support |
